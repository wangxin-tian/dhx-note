# OA

## 简介

OA 系统，用于管理公司内部的各种事务。
自动通知相关人员，提高工作效率。

## 功能

- 用户管理：管理公司内部的用户，包括添加、编辑、删除用户信息。
- 部门管理：管理公司内部的部门，包括添加、编辑、删除部门信息。
- 请假管理：员工可以申请请假，包括请假类型、开始时间、结束时间、请假原因等。
- 审批管理：管理员可以审批员工的请假申请，包括通过、拒绝等。
- 通知管理：管理员可以发布通知，包括通知标题、通知内容等。
- 公告管理：管理员可以发布公告，包括公告标题、公告内容等。
- 报表管理：管理员可以生成各种报表，包括请假报表、审批报表等。

## 技术栈

- 前端：Vue、Element UI
- 后端：Spring Boot、MyBatis
- 数据库：MySQL
- 部署：Docker
